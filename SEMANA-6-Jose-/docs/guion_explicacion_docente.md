# Guion de exposición

Este proyecto implementa un juego Tres en Raya con IA.  
La IA usa Minimax con Poda Alfa-Beta para jugar de forma óptima.

El participante ingresa su nombre y elige modo:
- Fácil
- Normal
- Imposible

En modo imposible, la IA no debe perder.

La cámara se integra con OpenCV.  
El navegador muestra la cámara y Python analiza capturas pequeñas.  
Cuando OpenCV detecta el rostro, se habilita el turno humano.
